# pve-nag-remove

Bash-скрипт для удаления всплывающего окна «No valid subscription» из веб-интерфейса Proxmox VE.

> ⚠️ **Только для домашних лабораторий и тестовых стендов.**  
> В production-среде используйте официальную подписку — это поддерживает проект и даёт доступ к стабильным репозиториям и поддержке.

---

## Что делает

Модифицирует файл `/usr/share/javascript/proxmox-widget-toolkit/proxmoxlib.js` так, что функция проверки подписки перестаёт вызываться при входе в UI. Перезапускает `pveproxy` для применения изменений.

---

## Возможности

- Проверяет количество вхождений сигнатуры перед патчем — не применяет если найдено несколько (небезопасно)
- Создаёт backup с временной меткой и SHA256-контрольной суммой
- Автоматически восстанавливает файл из backup при сбое
- Проверяет что замена реально произошла после `sed`
- Режим `--dry-run` — показывает что будет сделано без изменений
- Флаг `--yes` для использования в скриптах и автоматизации
- Поддерживает кастомный путь к файлу через `--file`

---

## Использование

```bash
# Скачать и запустить
curl -O https://raw.githubusercontent.com/zxsergeant/proxmox-scripts/main/pve-nag-remove.sh
chmod +x pve-nag-remove.sh
sudo ./pve-nag-remove.sh
```

```
Опции:
  -y, --yes       Без подтверждения (автоматический режим)
  -n, --dry-run   Показать план без изменений
  -q, --quiet     Минимальный вывод (только ошибки)
  -f, --file      Путь к файлу (если отличается от стандартного)
  -h, --help      Помощь
```

---

## После применения патча

Браузер кеширует JS-файлы — после перезапуска `pveproxy` нужно:

1. Открыть Proxmox UI в **приватном окне**, или
2. Сделать **hard refresh**: `Ctrl+Shift+R` (Windows/Linux) / `Cmd+Shift+R` (macOS)

---

## Тестирование

Тесты написаны на [bats-core](https://github.com/bats-core/bats-core).

```bash
# Установка bats
apt install bats        # Debian/Ubuntu
brew install bats-core  # macOS

# Запуск
bats pve-nag-remove.bats
```

32 теста покрывают: логику патча, создание и восстановление backup, разбор аргументов, интеграционные сценарии.

---

## Важно

Патч слетает при обновлении пакета `proxmox-widget-toolkit`. После `apt upgrade` скрипт нужно запустить повторно.

---

## Совместимость
